# TeamFinder Backend

## Необходимо установить
- Java 17+
- PostgreSQL 14+

## Запуск проекта
1. Открываем папку с проектом в командной строке
2. Вводим для пересборки проекта - gradlew clean build
3. Вводим для запуска проекта - gradlew :backend:run

## Проверка на работу
Сервер: http://localhost:8080/health
Пример ответа от сервера в виде JSON: {"status":"ok"}
